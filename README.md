# 🎓 Student Management System (React)

This is a React-based project for managing students, grades, and attendance.

## ✨ Features

- Add, edit, and delete student data
- Record subject-wise grades
- Track attendance
- Dashboard with charts

## 🛠 Tech Stack

- React.js
- Tailwind CSS
- Context API
- Chart.js

## 🚀 How to Run

```bash
npm install
npm start
